# **App Name**: Safeviate Manager

## Core Features:

- Student Profile Management: Manage student profiles, track training progress, and record flight hours.
- Instructor Scheduling: Schedule instructors and manage their availability for training sessions.
- Aircraft Maintenance Logging: Record and track all required maintenance with relevant procedures.
- Personnel Management: Manage personnel data, roles, and permissions within the academy.
- Safety and Quality Management: Manage safety protocols, quality assurance processes, and compliance checks.
- Permissions Management: Control user access and permissions within the application based on roles.
- AI Features: AI features through when needed
- Login with Guest Access: Provide a secure login screen with a guest login option for developers.
- Write to Multi-Tenant Database: Write data to a multi tenant data base
- No Mock Data: No mock data only live data. Only placeholder data in main pages of menus
- Clean Sidebar Menu: Clean side bar menu
- Color Customization: Ability to change Primary background and Accent colours via Hex number colour picker in Page Format Menu

## Style Guidelines:

- Primary colour: Sky blue (#7CC4F7), evokes a sense of flight and openness.
- Background colour: Very light blue (#EBF5FB), for a calm, neutral backdrop.
- Accent color: Soft teal (#63B2A7), provides subtle contrast.
- Body and headline font: 'Inter' (sans-serif) for a modern and objective feel.
- Use clear, minimalist icons representing aviation and educational elements.
- Maintain a clean and organized layout with clear sections for different functionalities.
- Incorporate subtle animations to guide the user and provide feedback, e.g., smooth transitions for data updates.
